# Congiguración

### Este proyecto es el cazcaron de un proyecto con el estandar de Webpack.

- Node JS

- Webpack

- Babel

### Comandos de instalación:
#### Node Modules
``
npm install
``
#### Build
``
npm run build
``
#### Build sin minificado
``
npm run build:dev
``
#### Ejecución del servidor en desarrollo
``
npm start
``
